# @buzzy/mobile — Buzzy Today Mobile App

React Native (Expo) cross-platform mobile app for iOS and Android.

## Stack
- **Framework:** React Native + Expo
- **Navigation:** React Navigation
- **State:** Zustand + TanStack Query
- **Language:** TypeScript

## Development

```bash
# From monorepo root
npm run dev --workspace=@buzzy/mobile

# Or directly
cd apps/mobile && npx expo start
```

## Key Features
- Vertical swipe story browser
- Story cards with headline, TL;DR, key takeaways, buzzy take
- Bookmark system
- Topic tag following
- Signal-driven personalization
