# @buzzy/web — Buzzy Today Web App

Next.js 14 (App Router) web application with SSR for SEO.

## Stack
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Rendering:** SSR for Permanent Articles, ISR for performance

## Development

```bash
# From monorepo root
npm run dev --workspace=@buzzy/web

# Or directly
cd apps/web && npm run dev
```

Runs on `http://localhost:3000` by default.
